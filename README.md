# JoyWeb2.1 — Joydeep Mitra Portfolio

## Files
- `index.html` + `styles.css` + `script.js` — split production structure. Open
  `index.html` via a local server (VS Code Live Server) so CSS/JS load.
- `work/kyndryl-bridge.html` — Kyndryl Bridge deep-dive case page.
- `joydeep-mitra-portfolio.html` — single self-contained file (CSS+JS inlined).
  Double-click to open directly, no server needed.

## Run in VS Code
Open folder → install "Live Server" → right-click index.html → Open with Live Server.
(Or just open joydeep-mitra-portfolio.html.)

## What's in this build
- Rotating hero headline: 24 AI/design quotes, one per load, no repeat until all shown.
- Evolution walker on the timeline thread: illustrated single-contour silhouettes —
  ape → hunched hominin → upright erectus → striding human → modern human, ending in
  the agentic neural network. Scrubbed by scroll position.
- Timeline corrected: 2000 Aesthetic Technologies, 2003 Web Spiders, through Kyndryl.
- Fonts load from Google Fonts (needs internet on first load; falls back gracefully).
